# Campus Food Habits Dashboard

## 📌 Project Description

The **Campus Food Habits Dashboard** is a simple web-based mini project that analyzes and displays student food habits using a dataset. The project uses a survey dataset that contains information about students' food preferences and the food delivery applications they commonly use.

The system reads the dataset from an Excel file and displays the information in an interactive dashboard. Users can search for a student and view their preferred food item and the delivery application they mostly use.

This project demonstrates how datasets can be integrated with web technologies to visualize and analyze real-world information.

---

## 🎯 Features

* Load and read dataset from an Excel file
* Display student data in a table
* Search student by name
* Show dashboard cards with food preference details
* Simple and user-friendly interface

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder in **Visual Studio Code**.
3. Install the **Live Server extension**.
4. Right-click on `index.html`.
5. Click **Open with Live Server**.

The project will open in your browser.

## 📊 Dataset

The dataset used in this project contains information about students such as:

* Student Name
* Branch
* Year of Study
* Most Ordered Food Item
* Preferred Food Delivery App

---

## 🔍 Search Function

Users can enter a student's name in the search box and click the search button. The system will display the student's food preferences and delivery app on the dashboard.
